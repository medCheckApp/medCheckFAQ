Como usar o MedCheck

Navegação

  1. Use o menu lateral para navegar entre as telas do aplicativo
  2. O aplicativo possui 5 telas principais
    1. Pacientes: Lista de pacientes cadastrados
    2. Itens de Checklists: Itens que podem ser utilizados dentro das checklists
    3. Checklists: Checklists criadas utilizando os Itens de Checklist e que podem ser utilizadas dentro dos pacientes
    4. Relatórios: Gereção de relatório de dados das Checklists por unidade, período de tempo e tipo de checklist
    6. Intruções: Instruções de uso para o aplicativo
  4. O botão de Logout do sistema se encontra no menu lateral

2. Itens de Checklist

Os Itens de Checklist são usados para criar novas Checklists Alguns Itens são pré-definidos pelo sistema e não podem ser deletados ou editados 2.1. Use o menu lateral para acessar a tela de Itens de Checklist 2.2. Use o icone no canto superior direito para acessar a tela de cadastro de Item de Checklist 2.3. Entre os dados do Item e aperte em salvar 2.4. Na tela de listagem aperte em um Item para poder editar suas informações 2.5. Na tela de listagem selecione um Item e arraste-o para a esquerda para visualizar o botão de deletar

3. Checklist

Para criar uma Checklist é preciso que existam Itens de Checklist já criados Algumas Checklists são pré-definidos pelo sistema e não podem ser deletadas ou editadas 3.1. Use o menu lateral para acessar a tela de Checklists 3.2. Use o icone no canto superior direito para acessar a tela de cadastro de Checklist 3.3. Entre os dados da Checklist 3.4. Na parte inferior da tela aperte em Selecionar Itens Médicos e selecione os Itens que deseja na Checklist que está sendo criada 3.5. Na tela de listagem aperte em uma Checklist para poder editar suas informações 3.6. Na tela de listagem selecione uma Checklist e arraste-a para a esquerda para visualizar o botão de deletar

4. Paciente

4.1. Use o menu lateral para acessar a tela de Pacientes 4.2. Use o icone no canto superior direito para acessar a tela de cadastro de Paciente 4.3. Entre os dados do Paciente 4.5. Na tela de listagem aperte em um Paciente para poder ver suas informações 4.6. Na tela de visualização das informações do Paciente, aperte em "Editar Paciente" para editar as informações do Paciente 4.7. Na tela de listagem selecione um Paciente e arraste-o para a esquerda para visualizar o botão de deletar

5. Checklist-Paciente

5.1. Use o menu lateral para acessar a tela de Pacientes 5.2. Na tela de visualização das informações do Paciente, aperte em "Editar Paciente" para editar as informações do Paciente 5.3. Aperte em "Adicionar Checklist" para poder adicionar uma Checklist a um paciente já existente 5.4. Na tela de Editar Paciente seleciona uma Checklist e arraste-a para a esquerda para visualizar o botão de deletar. Essa ação não vai deletar a Checklist, apenas remover a Checklist daquele paciente. 5.5. Para responder uma Checklist basta acessar as informações de um paciente, escolher a Checklist que deseja responder e marcar os Itens que foram respondidos. Depois use o botão "Salvar" no canto direito superior para finalizar a resposta da Checklist e retornar a tela de informações do paciente.
